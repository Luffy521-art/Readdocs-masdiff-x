# 分布预测模型：不确定性感知训练方案

## 损失函数

为了让模型能输出预测的不确定度，必须放弃传统的均方误差（MSE），改用**负对数似然损失（NLL）**。模型不再输出单一分数，而是输出分布参数：均值 $\mu$ 和对数方差 $v = \log(\sigma^2)$。

**平滑版优化公式：**
$$\mathcal{L}(\theta) = \frac{1}{2} \left( v + \frac{(\rho - \mu)^2}{\exp(v)} \right)$$

> **动态博弈机制：** > * **误差小（预测准）：** 模型主动缩小 $v$（方差），表现出“自信”。
> * **误差大（离群点）：** 为防止除以 $\exp(v)$ 导致惩罚爆炸，模型被迫增大 $v$，承认“我不确定”，从而有效避免网络崩溃。

## 训练流程

### 1. 数据划分
** 必须按 Prompt 级别进行交叉验证或划分。同一道 Prompt 对应的所有变体回复（及其隐状态/真实分数），必须集体划入训练集或测试集，严禁同一题跨集，以防数据泄露。

### 2. 特征预处理与拼接
在每个时间步 $t$，提取并拼接以下两部分作为模型的输入特征矩阵：
* **内部认知：** 大模型的隐状态 $h_t$（如 3584 维）。
* **外部规则：** PRM 过程奖励 $prm_t$（标量）。

### 3. 网络架构设计
* **聚合层：** 使用 1D-CNN 或 Mean-Pooling 将序列特征压缩为定长上下文向量。
* **双分支输出头：** 经过两层 MLP 后，分为两个独立线性映射分支，分别输出无激活函数的标量 $\mu$ 和 $v$。

### 4. 优化与早停
使用 AdamW 优化器和 NLL 损失函数进行反向传播，建议开启梯度裁剪以防初期 $v$ 震荡带来的梯度爆炸。通过监控验证集的 Test NLL 指标触发早停，防止模型对方差产生过拟合。
