# 基于大模型隐状态与 PRM 联合特征的 Rho 预测代理模型

## 1 任务定义
本文档描述一个基于大型语言模型原生隐状态（Hidden States）与过程奖励模型（PRM, Process Reward Model）密集打分联合编码的轻量级代理模型（Surrogate Model）。本模型学习的对象是一个包含双重先验信息的条件概率分布 $p(\rho | \tau_{hidden}, R_{prm}, M_{action})$。

在测试阶段，模型输出 Student-t 分布的三个核心参数 $(\mu, \sigma, \nu)$，并以分布中心作为最终代理打分（$\hat{\rho}$）的点预测：
$$\hat{\rho} = \mu$$

## 2 数据集特征
本次实验数据集从大模型的前向传播（Forward Pass）与 PRM 打分模块中无损截取。主要特征字段严格定义如下：

* **隐状态特征 ($\tau_{hidden}$)**: 大语言模型输出的高维语义张量，形状为 $[N_{sample}, Seq\_Len, Hidden\_Dim]$。在本实验中，$Hidden\_Dim = 2560$。
* **过程奖励打分 ($R_{prm}$)**: 针对生成步骤的密集奖励信号张量，形状为 $[N_{sample}, Seq\_Len, 1]$。
* **动作掩码 ($M_{action}$)**: 动作掩码张量，用于区分生成过程中的有效 Token 与 Padding 填充区域。
* **真实评分 ($\rho$)**: 每个个体的目标评估得分（Target Rho）。

## 3 模型设计
本代理模型采用极简融合架构，通过在特征维度上直接拼接（Concatenation）语义特征与 PRM 打分，最大化保留大模型的先验判别能力。

### 3.1 动态对齐与联合特征拼接
由于不同批次（Batch）中隐状态、PRM 数组与掩码的截断长度可能不一致，模型首先在序列维度上进行动态最小长度对齐。
随后，在特征维度（Feature Dimension）上拼接隐状态与 PRM，构成联合特征 $X_{i}$：
$$X_{i,j} = [\tau_{i,j}, R_{prm, i,j}]$$
其中 $X_{i,j}$ 的特征维度被扩充为 $2561$ 维（$2560 + 1$）。

### 3.2 掩码池化 (Masked Mean Pooling)
对于第 $i$ 个样本，利用动作掩码过滤无效的 Padding 干扰，对联合特征进行全局池化，得到定长的句向量表示 $h_i$：
$$h_i = \frac{\sum_{j=1}^{S} X_{i,j} \cdot M_{i,j}}{\max(\sum_{j=1}^{S} M_{i,j}, \epsilon)}$$
其中 $S$ 为序列长度，$\epsilon = 1e-5$ 用于保证数值稳定。

### 3.3 深度特征映射与 Student-t 预测头
池化后的联合高维特征 $h_i$ 被送入包含 Dropout（防止小样本过拟合）的两层非线性多层感知机（MLP）：
$$h_{out} = \text{MLP}(h_i)$$

预测头最终输出三个分布参数：
$$\mu = h_{out}[:, 0]$$
$$\sigma = \text{softplus}(h_{out}[:, 1]) + 1e-4$$
$$\nu = \text{softplus}(h_{out}[:, 2]) + 1.01$$
约束 $\sigma > 0$ 和 $\nu > 1$ 保证了分布的方差存在且具有物理意义。

## 4 损失函数设计

### 4.1 Student-t 条件预测分布
模型假设目标真实值 $\rho_{true}$ 满足联合条件下的 Student-t 分布：
$$\rho_{true} | \tau_{hidden}, R_{prm}, M_{action} \sim \text{StudentT}(\nu, \mu, \sigma)$$
相比于高斯分布（MSE 损失的基础假设），Student-t 分布具有更厚的尾部（Fat-tail），对极端异常打分（Outliers）表现出极强的鲁棒性。

### 4.2 负对数似然损失 (NLL)
训练的主损失函数采用 Student-t 的负对数似然损失：
$$\mathcal{L}_{NLL} = -\log p(\rho_{true} | \tau_{hidden}, R_{prm}, M_{action})$$
该损失同时优化预测的准确性（逼近均值 $\mu$）以及预测的置信度（自适应调整尺度 $\sigma$）。

## 5 实验结果与可视化分析
本次实验共迭代 30 个 Epoch（$Batch\_Size = 8, LR = 1e-4$）。根据生成的 Dashboard 与终端日志，模型表现出极佳的收敛与泛化能力。

### 5.1 损失函数收敛分析 (Train vs Test Loss)
* **表现**：从 Loss 演进曲线可以看出，Train Loss 与 Test Loss 均呈现平滑下降趋势。
* **结论**：**模型完美避开了过拟合。** Test Loss 始终低于或紧随 Train Loss，证明 Student-t NLL 在 320 条小样本数据集上展现了出色的正则化效果。

### 5.2 核心测试集指标演进 (Test Set Metrics)
本次实验重点考察回归任务的三大指标：决定系数（$R^2$）、Pearson 相关系数（$r$）与排序准确率（Pairwise Ranking Accuracy）。
* **排序准确率（核心指标）**：模型在引入 PRM 信号后，达到了**历史最高排序准确率 86.35% (0.8635)**，远超 0.7 的预定阈值。这意味着在强化学习（PPO）采样阶段，代理模型有极高的概率能正确指引策略进化的方向。
* **Pearson 相关系数**：随训练稳定在 **0.72** 左右，证明模型准确捕获了代码质量特征与最终打分之间的同向波动趋势。

### 5.3 不确定度收敛与预测散点分布 (Sigma & Scatter Plot)
* **模型置信度 (Mean Sigma)**：Student-t 不确定性曲线呈现出指数级下降。$\sigma$ 最终稳定在 0.05 以下，表明代理模型在联合 PRM 与隐状态特征后，预测置信度极高。
* **预测对齐度 (Predicted vs True Rho)**：在最优轮次的散点图中，预测点高度聚拢并沿 $y=x$ 理想基准线分布。

## 6 思考与后续改进方向

1. **模型设计的成功验证**：本次实验证明了**“大模型高维隐状态 + PRM 密集信号 + 轻量 MLP”**是一种极高效率的架构。PRM 提供了细粒度的步骤级奖励指导，而隐状态提供了全局语义，两者联合极大提升了模型的鉴别能力。
2. **后续改进（Data Scaling）**：虽然排序指标（86.35%）已极度优秀，但受限于 320 条样本的绝对信息量，$R^2$ 分数仍有提升空间。未来可将数据集规模无监督扩充至 2000 级别，进一步拉满绝对分值的刻度精度。
